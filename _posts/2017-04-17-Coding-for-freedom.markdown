## 起因

原因复杂，细说也不便，记得当初我的 leader 在饭后散步和我们说起他的一个阿里同事的座右铭，即本文标题，coding for freedom（不晓得他的这个同事现在如何了）。这句话倒是一直印在我脑海里，这倒不是说出国了就 free 了，现在也很 free 。只是说可以去尝试一些不一样生活，毕竟，人生还是有很多选择的不是吗。这里 coding 也只是一个手段，恰逢学了这项技能，而且还能谋个生，攒点学费钱（攒不起老婆本，杭州的房价...哎...）。还有一个就是如果出去了，我还能找个 Aeroclub 学学开飞机呀，这种 club，国内还真没听说过，想想还是很激动的，再考个私照，以后说不定就能自由飞了。虽然飞不了 Boeing737 这种超级空客，但是塞斯纳这种小飞机，肯定没有问题的。

15年开始，慢慢的就有了这些想法了。15年春节，先跟家里通了个气，说想出去读个书，探探口风，倒是没有明确拒绝，只不过担心我这么大了还没结婚（那会才25！）。2016年，其实刚开始是什么都不懂，去哪个国家，怎么去，是不是读书，读什么专业，都是完全不知所措的。慢慢查资料继续学英语（此时扇贝打卡应该有一年了吧）。当明白直接去当地找工作是不太可能的之后，留学便成了必然了。4-8月疯狂刷扇贝的原版书，最终刷了90多本(牛津书虫就有60本，都是很快就能看完的)。9月份开始看国家，定学校，找中介。刚开始是找的澳洲，毕竟大部分要留学都去的这里。但是澳洲硕士要两年，差不多要花40多万，而且留下来也不那么容易。学校 offer 都收到了。最后心里还是放弃了，转向新西兰，一年读书，然后找到工作就能拿 PR 的实在是太吸引人了。

国庆在家看了 [Jack Liu](http://www.jack-liu.com/) 的博客之后开始考虑新西兰，一番罗列和对比，并且咨询了一下已经过去了的人，并看了很多文章，如：[这篇](http://ourcoders.com/thread/show/3659/)，最后心里有了答案。申请了 Unitec 的 PGD。

**以下是之前写的**

国庆前三天在家仔仔细细的看了 New Zealand 的 PR 方案，发现，这可能是比澳洲还要有优势的选项。又联系了好几个已经拿到 NZ PR 的程序员，发现可行性很高，而且，性价比也很高。下面是我列的对比表格

上个对比图

| 项目 | 澳洲  | 新西兰 |
|:----------:|:----------:|:-----------:|
| 硕士年份  | 2 | 1 |
| 硕士区别 | Master(硕士学位)  | 可拿Diploma(硕士文凭)即: PGD |
| 读书总开销 | 40W+(学费28-30W) |  20W+(学费10W) |
| PR 条件 | 雅思4个7(主要难点) | 找到相关工作(主要难点) |
| PR 难度 | 四个7难度一般 | 据说一般总能找到 |
| PR | 没有永久PR(需5年内待满2年) | 永久PR(即使回国工作)|
| PR 优势 | 拿到澳洲PR=拿到新西兰 PR | 不等于 |
| 薪资待遇 | 35W(RMB)/年 起 | 25W(RMB)/年 起 |
| 无聊程度 | 不无聊 | 非常无聊 |

读书的总开销来说，新西兰直接秒杀澳洲了，如果去新西兰的话，自己应该能存到这么多，对 PR 条件来说，雅思4个7其实在读硕士期间拿到对我来说并不是太大问题（虽然确实很难，但是宝宝有信心）。新西兰的程序员工作，据他们反馈，虽然难找也小众，但是努力还是能找到的。还有一点就是 PR 的优势了，优势也相当明显，即便你拿了 PR，去其他国家也能长期工作（比如我还想去 Singapore）。

综上所述，新西兰选择更好，中介那边的1000块保证金可能拿回不来了，哈哈。

昨天收到了弗林德斯大学的 conditional offer 了；塔斯马尼亚大学需要回答他们一个问题，问题已提交；伍伦贡大学比较麻烦，需要一个 GTE 声明和一份 PS（表明 GAP year 超过2年的原因），反正都不去了，我也不想写了。
